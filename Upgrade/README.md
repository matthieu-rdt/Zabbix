## Scripts for upgrading Zabbix server

### To upgrade Zabbix server 5

\- upgrade_zabbix_server_to_5.sh

> #### First, fill in these variables, lines 20 to 21
>
> `backup_password`  
> `database_name`  
>
> #### As argument, you can use
>
> `ubuntu`  
> `debian`  
> `rhel`

### Example

```bash
./upgrade_zabbix_server_to_5.sh ubuntu
```
